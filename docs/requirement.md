# Snippet-Vault 요구사항 분석

## 1. 프로젝트 개요
Snippet-Vault는 개발자가 자주 사용하는 코드 스니펫을 저장, 검색, 필터링하고 빠르게 복사할 수 있도록 돕는 웹 애플리케이션이다.

## 2. 개발 목적
기존 메모장형 도구보다 코드 재사용성과 검색 효율을 높이고, 실무에서 자주 사용하는 코드를 자산처럼 관리할 수 있도록 한다.

## 3. 핵심 기능
1. 코드 스니펫 저장
2. 코드 스니펫 목록 조회
3. 키워드 검색
4. 언어별 필터링
5. 태그별 필터링
6. 원클릭 복사
7. 코드 하이라이팅

## 4. 사용자 시나리오
- 사용자는 새로운 코드 스니펫을 저장할 수 있다.
- 사용자는 저장된 스니펫을 목록으로 확인할 수 있다.
- 사용자는 언어, 태그, 키워드로 원하는 스니펫을 찾을 수 있다.
- 사용자는 버튼 클릭 한 번으로 코드를 복사할 수 있다.

## 5. 기대 효과
- 자주 사용하는 코드의 재사용성 향상
- 개발 생산성 향상
- 코드 관리의 체계화
